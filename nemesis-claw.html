Griffe Némésis

Kill Team 2024 · Recherche et destruction / Infiltration
Chaos Hereticus Astartes Night Lords
Composition
1 Visionnaire (chef) + 5 agents choisis parmi :
Épouvanteur · Tireur · Tireur Lourd · Crisseur · Volepeau · Ventrilokar · Guerrier

Hormis les agents Guerrier, votre kill team ne peut inclure chacun des agents de cette liste qu'une seule fois.
Règles de Faction
Drapé de nuit

Quand un agent ennemi tire sur un agent GRIFFE NÉMÉSIS ami, cet agent ami est masqué si les deux critères suivants se vérifient : il est à plus de 8" des agents ennemis dont il est visible ; il a un terrain Lourd à portée de contrôle, ou n'importe quelle partie de son socle est sous un terrain de type Promontoire.

Astartes

À chaque activation d'agent GRIFFE NÉMÉSIS ami, il peut effectuer soit deux actions Tirer, soit deux actions Combattre. S'il s'agit de deux actions Tirer, vous devez choisir un pistolet bolter, un bolter ou un pistolet bolter à lunette pour au moins l'une d'elles.
Chaque agent GRIFFE NÉMÉSIS ami peut contre-attaquer quel que soit son ordre.

Subterfuges Stratégiques
1PC
Retour aux Ténèbres

1 agent GRIFFE NÉMÉSIS ami peut immédiatement effectuer une action Battre en Retraite ou de Repositionnement gratuite, mais il doit finir ce mouvement avec un terrain Lourd à sa portée de contrôle, ou avec n'importe quelle partie de son socle sous un terrain de type Promontoire. De plus, il ne peut pas se déplacer de plus de 4" pendant cette action et il ne peut pas finir ce mouvement en se trouvant plus près qu'il ne l'était des agents ennemis.

1PC
Nous sommes venus vous prendre

Quand un agent GRIFFE NÉMÉSIS ami est activé, si la première action qu'il effectue pendant cette activation est l'action Charger, quand il finit son déplacement à cette action, vous pouvez infliger D3 dégâts à un agent ennemi à sa portée de contrôle.

1PC
Thermographe

Quand vous choisissez une cible éligible pour un agent GRIFFE NÉMÉSIS ami, les agents ennemis à 6" de lui ne peuvent pas utiliser le terrain Léger comme couvert. Bien que cela puisse permettre de cibler de tels agents (à condition qu'ils soient visibles), cela ne les prive pas de leur sauvegarde de couvert (le cas échéant).

1PC
La chasse noire

Quand un agent GRIFFE NÉMÉSIS ami tire sur, combat ou riposte contre un agent ennemi blessé, vous pouvez relancer un de vos dés d'attaque.

Subterfuges d'Affrontement
1PC
Combattant déloyal

Utilisez ce subterfuge quand un agent GRIFFE NÉMÉSIS ami riposte, au début de l'étape Résoudre les Dés d'Attaque. Vous pouvez résoudre une de vos réussites avant l'ordre normal. Dans ce cas, vous ne pouvez pas résoudre d'autre réussite pendant cette séquence.

1PC
Hurlement Vox

Utilisez ce subterfuge quand votre adversaire est censé activer un agent ennemi visible par un agent GRIFFE NÉMÉSIS ami. Jetez un D6 ; si le résultat est supérieur à la LPA de l'agent ennemi, l'adversaire ne peut pas l'activer ; si inférieur ou égal, ce subterfuge n'est pas utilisé, le PC est récupéré et vous ne pouvez plus utiliser ce subterfuge à ce tournant.

1PC
Propension au meurtre

Utilisez ce subterfuge après qu'un agent GRIFFE NÉMÉSIS ami neutralise un agent ennemi à sa portée de contrôle. L'agent ami peut immédiatement effectuer une action Charger ou Sprinter gratuite (pour l'action Charger gratuite, il ne peut pas se déplacer de plus de 3").

1PC
Mort au faux Empereur

Utilisez ce subterfuge après avoir jeté les dés d'attaque contre un agent ennemi avec le mot-clé IMPERIUM. Les armes de cet agent ami ont la règle Inexorable jusqu'à la fin de cette séquence ; si l'agent ennemi a également le mot-clé ADEPTUS ASTARTES, les armes ont la règle Implacable à la place.

PresciencePsychique

À l'étape Préparer de chaque phase de Stratégie, vous gagnez D3 points de Prescience. À la fin de chaque tournant, défaussez vos points. Vous pouvez les dépenser à la phase d'Affrontement (une fois par tournant chacun) :

  • Pressentiment : Quand c'est votre tour d'activer un agent ami, vous pouvez dépenser 1 point pour sauter cette activation.
  • Présage : Quand un dé d'attaque inflige des Dégâts Normaux à cet agent, vous pouvez dépenser 1 point pour ignorer les dégâts infligés.

Vous ne pouvez pas gagner ni dépenser de points de Prescience si cet agent est neutralisé.

Équipement de Faction
Chaînes de sujétion

Quand un agent ennemi va effectuer l'action Battre en Retraite alors qu'il est à portée de contrôle d'un agent GRIFFE NÉMÉSIS ami, s'il n'y a pas d'autre agent ennemi à portée de contrôle de l'agent ami, vous pouvez utiliser cette règle. Jetez deux D6, ou un D6 si l'agent ennemi a une caractéristique de PV plus élevée. Si au moins un dé donne 4+, l'agent ennemi ne peut pas effectuer cette action, aucun PA n'est dépensé et vous ne pouvez plus utiliser cette règle pendant ce tournant.

Brouilleurs de transmissions

Quand un agent ennemi est à 3" d'un agent GRIFFE NÉMÉSIS ami, la caractéristique de LPA de l'agent ennemi ne peut pas être améliorée. Notez que cela n'affecte pas les caractéristiques de LPA déjà modifiées.

Trophée macabre

Une fois par bataille, quand un agent GRIFFE NÉMÉSIS ami neutralise un agent ennemi à 2" de lui, cet agent ami gagne un pion de Trophée Macabre (s'il n'en a pas déjà un). Quand un agent ami avec ce pion est à 2" et visible d'un agent ennemi, soustrayez 1 à la caractéristique d'Attaques des armes dudit agent ennemi.

Peau dépecée

Quand un agent ennemi tire sur, combat ou riposte contre un agent GRIFFE NÉMÉSIS ami à 2" de lui, votre adversaire ne peut pas relancer ses dés d'attaque dont le résultat est 1.
