Sécurité / Recherche et destruction
Deathwatch Imperium Adeptus Astartes
Composition

5 agents DEATHWATCH choisis dans la liste suivante : SERGENT DU GUET, ÉGIDE, MAÎTRE ESCRIMEUR, BOMBARDIER, BRÈCHEUR, DÉMOLISSEUR, DISRUPTEUR, TIREUR, DÉCAPITEUR, HORDICIDE, TIREUR D'ÉLITE.

Votre kill team ne peut inclure qu'un seul de chaque agent de cette liste, et ne peut inclure qu'un seul agent GRAVIS.

Règles de faction
Vétérans de l'Adeptus Astartes

Pendant chaque activation d'un agent DEATHWATCH ami, celui-ci peut effectuer soit deux actions Tirer, soit deux actions Combattre. Dans le cas de deux actions Tirer, 1PA supplémentaire doit être dépensé pour la deuxième action si les deux actions utilisent un lance-grenades auxiliaire, un canon Frag, un incinérateur à plasma lourd, un bolter lourd Infernus, un pistolet à plasma ou un fusil bolter Stalker, ou si une bombe à fusion est choisie pour une des deux actions.

Munitions spéciales

Une fois par tournant, quand un agent DEATHWATCH ami effectue l'action Tirer, lors de l'étape Choisir l'Arme, vous pouvez utiliser cette règle. Choisissez 1 des règles d'arme suivantes : Déflagration 1", Dévastatrice 1, Létale 5+, Perforante Critiques 1, Saturation, Fatale.

Subterfuges Stratégiques
1 PC
Tactiques de mission

Choisissez Dissimulation ou Engagement. Quand un agent DEATHWATCH ami tire sur ou combat contre un agent ennemi qui a cet ordre, les armes de cet agent ami ont la règle Équilibrée.

1 PC
La longue veille

Chaque fois qu'un agent tire sur un agent DEATHWATCH ami entièrement sur votre territoire, vous pouvez relancer un de vos dés de défense.

1 PC
Le bouclier mortel

Quand un agent DEATHWATCH ami est entièrement sur le territoire adverse, les Dégâts Normaux de 4 ou plus lui infligent 1 dégât de moins.

1 PC
Et ils ne connaîtront pas la peur

Vous pouvez ignorer les modifications aux caractéristiques des agents DEATHWATCH amis dues au fait d'être estropié (y compris pour les caractéristiques de leurs armes).

Subterfuges d'Affrontement
1 PC
Ne tolérez pas le xéno

Utilisez ce subterfuge d'affrontement après avoir jeté vos dés d'attaque pour un agent DEATHWATCH ami, s'il tire sur ou combat contre un agent qui n'a pas le mot-clé CHAOS ou IMPERIUM. Vous pouvez relancer un ou plusieurs de vos dés d'attaque.

1 PC
Traçage d'auspicateur

Utilisez ce subterfuge d'affrontement quand un agent DEATHWATCH ami contre-attaque, avant qu'il effectue la moindre action. Vous pouvez changer son ordre.

1 PC
Scan d'auspex avancé

Utilisez ce subterfuge d'affrontement quand un agent DEATHWATCH ami effectue l'action Tirer. Jusqu'à la fin de l'activation/contre-attaque, ses armes de tir ont la règle Saturation et les agents ennemis ne peuvent pas être masqués.

1 PC
Physiologie transhumaine

Utilisez ce subterfuge d'affrontement quand un agent tire sur un agent DEATHWATCH ami, à l'étape Jeter les Dés de Défense. Vous pouvez conserver 1 de vos réussites normales comme une réussite critique.

Équipement de faction
Armes digitales

Une fois par tournant, quand un agent DEATHWATCH ami effectue l'action Combattre, au début de l'étape Jeter les Dés d'Attaque, vous pouvez utiliser cette règle. Dans ce cas, infligez 1 dégât à l'agent ennemi de cette séquence.

Servoséide scrutavore

Une fois par tournant, pendant l'activation d'un agent DEATHWATCH ami, vous pouvez utiliser cette règle. Dans ce cas, pendant cette activation, cet agent peut effectuer une action de mission pour 1PA de moins. Avoir un agent ennemi à sa portée de contrôle n'empêche pas cet agent ami d'effectuer cette action de mission. Toutefois, dans un tel cas, après qu'il a effectué l'action de mission, votre adversaire et vous tirez au dé. Si votre adversaire gagne, vous ne pouvez plus utiliser cet équipement pour le restant de la bataille.

Brassards bioradars de sanctus-v

Une fois pendant chaque activation d'un agent DEATHWATCH ami, avant qu'il n'effectue une action ou après qu'il a effectué une action, s'il n'est à portée de contrôle d'aucun agent ennemi, vous pouvez utiliser cette règle. Dans ce cas, choisissez 1 des choses suivantes :

  • L'agent ami récupère D3 PV perdus.
  • Retirez tout changement à la caractéristique de LPA de cet agent ami.
  • Retirez un des pions suivants qu'a cet agent ami (avant de résoudre les effets d'activation de ce pion, le cas échéant) : Fragment Neutronique, Poison, Terrochimique.
Réserve de munitions

Une fois par bataille, vous pouvez utiliser la règle de faction Munitions Spéciales pour deux actions Tirer pendant un tournant, mais vous devez choisir différentes règles d'arme pour chacune de ces utilisations.
